# 나의 포트폴리오

#### 📝 개요
+ KaKaoMap API와 OpenWeather API를 사용해서 날씨알리미 프로젝트를 해보았습니다.
+ 현재기온, 시간별 및 날짜별 기온, 위치별 기온의 기능을 만들었습니다.
+ TanStack Query라이브러리를 사용하여 상태관리를 하였습니다.

#### ⚒ 사용 기술
+ HTML, CSS, JAVASCRIPT, REACT
+ React-icons, TanStack Query, styled-components
+ FirebaseAPI

#### 🔗 공유 링크
